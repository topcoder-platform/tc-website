This is a fully functional demonstration that uses component technology developed entirely by TopCoder Software using the TopCoder Software Distributed Development Environment.

What you will see and do
We ask that while going through the TestDrive!, that you take note of what components are in use. You will find a listing in the right blue column.

Again, you are using actual software so please be sure that the event you create and the people you send it to is not of offensive nature.


Each of the following can be found in the Component Sales Catalog.
Event Engine: This component processes messages and triggers associated events.
Email Engine: This performs the functionality that its name implies.
Document Generator: Merges data and templates to create customized documents.
When bundled together, the Bulk Mailer, Document Generation, Event Engine and Configuration Manager components create the TopCoder Software Email Component Assembly.

This component assembly is an extremely efficient set of functionality to create, schedule and mail customized documents in an efficient manner.
