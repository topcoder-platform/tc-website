Affordable Software Solutions

TopCoder Software components are purchased through TopCoder Software Component Subscriptions. The Component Subscription model offers customers the flexibility to choose only those components that suit their needs. In addition, customers are encouraged to collaborate with TopCoder Software so that their requirements can be integrated into new components when they are added to the catalog. TopCoder Software Customer Collaboration ensures that the direction of TopCoder Software's component development pipeline meets the evolving needs of its customers.

Each component in the TopCoder Software Component Catalog is assigned a unit value that corresponds to the work effort required to develop the component. Customers have the flexibility to purchase subscriptions for the number of component units that they need. With a subscription they receive components equal to the number of units in their subscription cap, support hours equal to the number of components in their subscription cap and access to component upgrades. Volume discounts are provided for higher-level subscriptions.

Subscriptions, as of September 5, 2002 start at $5,000 for five units and five hours of support. Volume discounts will apply at higher subscription levels. Subscription prices may change without notice.

One of the reasons TopCoder Software components are such a great value is that we practice the re-use that we preach. Our components are typically developed using existing TopCoder Software base components. Customers are never charged for an individual base component more than once. The cost of the purchased component includes all base components it uses. For instance, a six CPU license for our file conversion component represents six component units. The file conversion component was developed using the configuration manager base component. The configuration manager base component is included with the file conversion component purchase. When that customer purchases the email engine component, which also uses the configuration manager base component, they pay only one unit (instead of three) since the configuration manager base component (two unit value) has already been purchased.
