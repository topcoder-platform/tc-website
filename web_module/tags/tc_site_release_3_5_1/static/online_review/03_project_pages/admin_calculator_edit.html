Overview
The Financial Calculator provides basic and complex financial calculations. These calculations include, but are not limited to, fixed-payment computation (mortgage calculator), net present value and future value. The Financial Calculator component can be used in any application requiring financial calculations. This can range from mortgage payment to accounting software.
